# Sync Ethers Go

Sync onchain events to mongodb over a easy to use REST API.

---

## HOW TO USE

- ### With docker
  - Install Docker and docker compose
  - docker-compose up
- ### Others
  - Install Go
  - Vendor local dependencies
  - Run using go run
  - (optional) Install Air via Go Mod, run using air for hot reload.
